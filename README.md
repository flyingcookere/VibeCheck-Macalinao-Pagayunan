## Features

### Backend API
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/fortune` | GET | Returns a random fortune message |
| `/api/joke` | GET | Returns a random developer joke |
| `/api/vibe?mood=happy|tired|stressed` | GET | Returns a vibe based on the mood |
| `/api/smash` | POST | Increases the in-memory smash counter by 1 |
| `/api/smashes` | GET | Returns the current smash counter |
| `/api/secret?code=411L` | GET | Returns a secret message if the correct code is provided |

## Setup / Installation

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd VibeCheck-Macalinao-Pagayunan
````

### 2. Install backend dependencies

```bash
cd backend
npm install
npm init -y
npm install express cors

```

* Installs: `express` and `cors`

### 3. Run the backend server

```bash
node index.js
```

* The server runs at: `http://localhost:3000`

### 4. Open the frontend

#### Option 1: Directly in browser

* Open `frontend/index.html` in your browser.

#### Option 2 (recommended): Using a local server

```bash
cd frontend
npx serve .
```

* Open the URL provided by `serve` (e.g., `http://localhost:5000`) in a browser.
* This ensures all fetch requests work correctly.
